# Dokucraft Hytale

Dokucraft Light Support for Hytale

This is a Work-In-Progress, so expect changes and possibly some Bugs

If you have any Issues/Suggestions, please Submit them in [Issues](https://github.com/Dokucraft/Dokucraft-Hytale/issues)

#### Installation:
* Download [Dokucraft-Hytale.zip](https://github.com/Dokucraft/Dokucraft-Hytale/releases/download/Latest/Dokucraft-Hytale.zip)

* Extract it and Place `Dokucraft-Hytale` (A Folder with `manifest.json` inside) into `%appdata%\Hytale\UserData\Saves\WORLD_NAME\mods`
